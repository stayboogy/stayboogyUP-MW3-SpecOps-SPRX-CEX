```javascript
a) extract the zip

b) copy "stayboogyUP.sprx" to "hdd0\tmp\stayboogyUP.sprx"

c) copy "EBOOT.BIN" to "hdd0\game\BLUS30838\USRDIR\EBOOT.BIN"
```
